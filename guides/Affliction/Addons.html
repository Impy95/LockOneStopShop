Addons:

  • Elv UI This is a great UI skin package, super clean, and very customizable. It also has a very nice module for customizing the boss unit frames for tracking dots easier.
  • Weak Auras 2 This is super duper helpful with tracking any sort of event / aura / thing.
  • Doom Shards Handy UI for tracking shards and when you can expect a flood of shards. Has some "Rough predictions" of when you will likely get a shard from Agony. This version of Doomshards has been modified to work in tandem with the Soul Shard bit change that comes into effect in 7.2.5.
  • Ipse Shards As an alternative to DoomShards there is a new Weak Aura by an amazing coder named Ipse who is replicating the functionality of DoomShards into a Weak Aura aptly called Ipse Shards which can be found by clicking on the link above. Macros

If you would like to submit a macro for this section, please PM me on Discord.

How to make Demonic Circle work like PRE legion:
#showtooltip Demonic Circle
/cancelaura Demonic Circle
/cast Demonic Circle

Cancel Burning Rush when stationary (IE: Draining) This will also work nicely with Unstable Affliction.
#showtooltip Drain Soul
/cancelaura Burning Rush
/cast Drain Soul

Alt/Shift/Ctrl modifier focus macro. If you hold down the selected mod key, the macro will cast the spell at your focus rather than your current target.
#showtooltip Agony
/cast [mod:alt,@focus][] Agony

Mouseover Dots
#showtooltip Agony
/cast [target=mouseover,harm][] Agony
