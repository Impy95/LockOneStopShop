Disclaimer!

The Raw Value sims do not replace any gear, and assumes everything is 945 ilevel. They are intended to show the raw value of the set bonus itself.
Rules used for the realistic gear sims:
  • All tier items are the base ilevel for that tier. T19M = 905. T20M = 930. T21H = 945. T21M = 960.
  • Legendaries are 1000 ilevel.
  • All other items use heroic antorus base ilevel (945).
  • Helm/Belt for 4 piece setups.
  • Swaps Helm/Belt to Belt/Netherlord when running 6 piece combinations.
  • Uses SL if >1% ahead of SC.
