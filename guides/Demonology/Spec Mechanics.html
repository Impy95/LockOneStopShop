Demonic Consumption

Optimal use of Demonic Consumption involves some forethought or the use of a WeakAura/Addon to track the power gain from summoning Demonic Tyrant based on how many Wild Imps are currently summoned.
The power gained from consuming your Wild Imps is based on how much remaining energy they have:

With this in mind you should be looking to summon optimally Demonic Tyrant with two sets of Wild Imps out that are summoned back to back, one of which will be at full energy. The cast sequence would look something like this:
  1. Build to 5 Soul Shards
  2. Cast Hand of Gul'dan
  3. Build to 3 Soul Shards
  4. Cast Hand of Gul'dan
  5. Cast Summon Demonic Tyrant

Implosion

Implosion is, as always, a topic of debate on how to best use the ability. Contrary to Legion Implosion there is a bit more of rotation, if you will, depending on if you have a Demonic Core stack or not.
We can explore these situations a bit more in details here:

If you have 2 or more stacks of Demonic Core you...

  • Build to 5 Soul Shards
  • Cast Hand of Gul'dan
  • Build 4 Soul Shards using Demonbolt
  • Cast Hand of Gul'dan
  • Build to 3 Soul Shards using Demonbolt
  • Cast Hand of Gul'dan
  • Cast Shadow Bolt
  • Cast Shadow Bolt
  • Cast Implosion
If you have 1 or 0 stacks of Demonic Core you...
  • Build to 5 Soul Shards
  • Cast Hand of Gul'dan
  • Build to 3 Soul Shards
  • Cast Hand of Gul'dan
  • Cast Shadow Bolt or Demonbolt if you have a stack of Demonic Core
  • Cast Shadow Bolt
  • Cast Implosion
