Nether Portal Single Target Priority Rotation

This priority rotation is assuming you are using one of the default single target talent selections.

  1. Pre-cast Demonbolt
  2. Cast Demonic Strength if talented, on cooldown
  3. Cast Nether Portal if talented, on cooldown - Enter the Nether Portal Rotation
    1. Cast Call Dreadstalkers
    2. Cast Shadow Bolt
    3. Cast Grimoire: Felguard
    4. Cast Shadow Bolt
    5. Cast Summon Vilefiend
    6. Cast Shadow Bolt
    7. Cast Hand of Gul'dan
    8. Cast Shadow Bolt
    9. Cast Hand of Gul'dan
    10. Cast Summon Demonic Tyrant
    11. Cast Shadow Bolt
    12. Cast Hand of Gul'dan
    13. Cast Shadow Bolt
    14. Cast Hand of Gul'dan
    15. Cast Shadow Bolt
    16. Cast Hand of Gul'dan
  4. Cast Summon Demonic Tyrant on cooldown
  5. Build to 3/4 Soul Shards, depending if Demonic Calling Procs or not
  6. Cast Call Dreadstalkers
  7. Cast Hand of Gul'dan
  8. Cast Demonbolt as needed to not overcap
  9. Cast Summon Vilefiend on cooldown
  10. Fill with Demonbolt if you have available stacks of Demonic Core - alternately fill with Shadow Bolt

Sacrificed Souls or Demonic Consumption Single Target Priority Rotation

Opening priority:
  1. Pre-cast Demonbolt
  2. Cast Demonic Strength, if talented
  3. Cast Summon Vilefiend, if talented
  4. Cast Call Dreadstalkers
  5. Cast Hand of Gul'dan
  6. Cast Summon Demonic Tyrant
Regular priority:
  1. Cast Summon Demonic Tyrant on cooldown
  2. Cast Demonic Strength on cooldown, if talented
  3. Cast Summon Vilefiend on cooldown, if talented
  4. Cast Call Dreadstalkers on cooldown
  5. Cast 3 Soul Shards Hand of Gul'dan whenever possible
  6. Cast Demonbolt or Shadow Bolt to generate Soul Shards

Two+ Target Priority Rotation

  1. Pre-cast Demonbolt
  2. Cast Summon Vilefiend if talented, on cooldown
  3. Cast Grimoire: Felguard if talented, on cooldown
  4. Cast Call Dreadstalkers, on cooldown
  5. Cast Summon Demonic Tyrant, on cooldown with as many of the above cooldown demons and Wild Imps out
  6. Cast Doom on all possible targets, if talented and mobs will live for at least 30s
  7. Build to 3 Soul Shards
  8. Cast Hand of Gul'dan
  9. If targets are stacked already enter Implosion Rotation
    With a stack of Demonic Core:
    • Build to 5 Soul Shards
    • Cast Hand of Gul'dan
    • Build 4 Soul Shards using Demonbolt
    • Cast Hand of Gul'dan
    • Build to 3 Soul Shards using Shadow Bolt
    • Cast Hand of Gul'dan
    • Cast Implosion
    Without a stack of Demonic Core:
    • Build to 5 Soul Shards
    • Cast Hand of Gul'dan
    • Build to 3 Soul Shards
    • Cast Hand of Gul'dan
    • Cast Implosion
  10. Fill with Demonbolt if you have available stacks of Demonic Core - alternately fill with Shadow Bolt
